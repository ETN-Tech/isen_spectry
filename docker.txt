Créer une image (objet) à partir d'un Dockerfile dans le répertoire courant et le nommer "paul"
docker build . --tag="http_serv"

Voir la liste des images (objets) disponibles
docker images

Supprimer une image
docker rmi id

Créer un container sur la base d'une image (instancier) et l'exécuter
docker run http_serv:latest

idem et la lier interactivement à la console
docker run -ti http_serv:latest /bin/sh

idem et lier le port 8080 de l'host avec le port 5000 du container et la faire tourner en daemon
docker run -dp 6000:5000 --env-file=http_env --env-file=../db_env http_serv:latest

Arrêter container / daemon
docker stop id

Lister les container
docker ps -a

Nettoyer les arborescences : supprimer containers inactifs et images inutilisées
docker system prune

reste à voir : 
docker create
docker start
